# TailwindCSS2 (v0.1.0)

- **OOCSS** is the preferred methodology (Object-Oriented CSS)
- Classes should be in recursive alphabetical order
- use inline class for up to 3 utility classes

Order of properties:

```css
.class {
  /* Display (Flex, grid, etc.) */

  @apply /* Margin and padding */
    /* Background */
    /* Shadows */;
}
```
